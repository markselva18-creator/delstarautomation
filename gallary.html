To show multiple photos when clicking on a project tile, you can implement a **Modal Lightbox Gallery**. When a user clicks a project tile, a modal overlay pops up containing a slideshow/carousel of images specific to that project. Here is a clean, responsive, and reusable implementation using HTML, CSS, and vanilla JavaScript. ### 1. The HTML Structure Add a `data-images` attribute to your project tiles containing a comma-separated list of image URLs. Then, add the modal structure at the bottom of your page. ```html
Assembly Line

EV Battery Module Test System

Click to view gallery

×
``` --- ### 2. The CSS Styling This handles the dark overlay layout, centers the active image, and styles the navigation buttons. ```css /* Modal Background */ .modal { display: none; position: fixed; z-index: 1000; left: 0; top: 0; width: 100%; height: 100%; background-color: rgba(0, 0, 0, 0.9); align-items: center; justify-content: center; } /* Modal Content Wrapper */ .modal-content { position: relative; max-width: 80%; max-height: 80%; display: flex; align-items: center; justify-content: center; } .image-container img { max-width: 100%; max-height: 80vh; object-fit: contain; border-radius: 4px; } /* Close Button */ .close-btn { position: absolute; top: 20px; right: 35px; color: #fff; font-size: 40px; font-weight: bold; cursor: pointer; transition: 0.3s; } .close-btn:hover { color: #bbb; } /* Navigation Buttons */ .nav-btn { cursor: pointer; position: absolute; top: 50%; transform: translateY(-50%); background: rgba(255, 255, 255, 0.2); color: white; border: none; padding: 16px; font-size: 24px; border-radius: 50%; transition: 0.3s; } .nav-btn:hover { background: rgba(255, 255, 255, 0.5); } .prev-btn { left: -60px; } .next-btn { right: -60px; } ``` --- ### 3. The JavaScript Logic This script reads the image array from the clicked tile, opens the modal, and manages the image cycling. ```javascript document.addEventListener("DOMContentLoaded", () => { const tiles = document.querySelectorAll(".project-tile"); const modal = document.getElementById("galleryModal"); const modalImg = document.getElementById("modalImage"); const closeBtn = document.querySelector(".close-btn"); const prevBtn = document.querySelector(".prev-btn"); const nextBtn = document.querySelector(".next-btn"); let currentImages = []; let currentIndex = 0; // Open modal on tile click tiles.forEach(tile => { tile.addEventListener("click", () => { const rawImages = tile.getAttribute("data-images"); if (rawImages) { currentImages = rawImages.split(",").map(img => img.trim()); currentIndex = 0; showImage(currentIndex); modal.style.display = "flex"; } }); }); // Update Image Source function showImage(index) { modalImg.src = currentImages[index]; } // Next/Prev Controls nextBtn.addEventListener("click", (e) => { e.stopPropagation(); // Prevents closing modal currentIndex = (currentIndex + 1) % currentImages.length; showImage(currentIndex); }); prevBtn.addEventListener("click", (e) => { e.stopPropagation(); currentIndex = (currentIndex - 1 + currentImages.length) % currentImages.length; showImage(currentIndex); }); // Close Modal Actions closeBtn.addEventListener("click", () => modal.style.display = "none"); modal.addEventListener("click", (e) => { if (e.target === modal) modal.style.display = "none"; }); // Keyboard Arrow Support document.addEventListener("keydown", (e) => { if (modal.style.display === "flex") { if (e.key === "ArrowRight") nextBtn.click(); if (e.key === "ArrowLeft") prevBtn.click(); if (e.key === "Escape") modal.style.display = "none"; } }); }); ```
